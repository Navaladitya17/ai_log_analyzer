Notes

  • Upload your system or security log in CSV format.
  • Files larger than 30 MB will be analyzed in chunk mode for efficiency.
  • Analysis runs locally — your data never leaves your computer.

About This Project

AI LOG ANALYZER is an advanced cybersecurity tool built to detect unusual activity or anomalies inside system logs using AI-based isolation models. It helps identify potential intrusions, network breaches, or misconfigurations before they cause damage.

Built with Python, Flask, and Machine Learning, it performs anomaly detection and visual analytics completely offline for maximum data privacy.
